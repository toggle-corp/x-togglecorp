Notifica

Notifica is an android application, with an accompanying web app, built for students and teachers for increasing classroom productivity. It provides students and teachers a shared medium for notices, assignments and class schedule, keeping users up to date about all sorts of classroom activities.

Flipped

Flipped is a platform-adventure game in a world of mirrors, which when crossed lead to alternate realities with conflicting gravities. It is a 2d mobile game with side-scroller view. The player has to guide a boy through a world where the obstacles are ‘Mirrors’ such that when the protagonist encounters a mirror, he is ported to a world where everything gets flipped.

Microphone

Microphone is a highly customizable audio recording application. It provides users with all available settings that their device support and allows for the best voice recording results.

Miti

Miti is a nepali calendar application with minimum hassle. It provides clean interface for displaying the Bikram Sambat calendar along with tithi. It also comes with some pretty nice widgets for home screen.

Chrono

Chrono is an application to track time of tasks in team projects. It uses the collected data to present reports with graphs and charts.

Remote Control

Remote control is a client server based application that allows users to control keyboard and mouse inputs in their computers from their mobile phones.
